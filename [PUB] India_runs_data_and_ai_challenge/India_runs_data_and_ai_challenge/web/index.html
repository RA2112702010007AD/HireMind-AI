🧠 HireMind-AI System Architecture

Composite Ranking Logic

HireMind-AI scores candidates by mapping their structured profiles against specific job description requirements using a multi-factor weighting engine:

  • Role Score (40%): Assesses current and past job titles. AI/ML engineers receive full points, general tech gets a minor discount, and irrelevant titles (HR, Marketing) are heavily penalized.
  • Experience Alignment (20%): Matches target founding-level seniority (5-9 YOE) and penalizes excessive or junior bands.
  • Skills Trust Scoring (30%): Blocks keyword stuffers. Replaces simple keyword counting with a weighted trust index based on skill use duration and peer endorsements:
    Trust = (duration_months / 12) * ln(e + endorsements)
  • Plain-Language Extraction (20%): Searches career descriptions for key projects (e.g., search indexing, recommendation engines) to identify great candidates who don't load trendy buzzwords.
  • Behavioral Modifiers (Multipliers): Real-time platform signals (responsiveness, notice period, location willingness, GitHub contributions) are used as multiplicative factors.

Honeypot Filter

All candidates run through an integrity check to automatically identify and drop trap profiles that contain logical contradictions (e.g. expert proficiency with 0 months used, or job durations exceeding total experience).
